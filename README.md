## Projetos em Destaque

### ENNIE - Sistema de Gestão Acadêmica com IA
**Tecnologias:** Node.js, Express, Socket.IO, JavaScript, HTML5, CSS3

Sistema completo de gestão acadêmica desenvolvido para otimizar processos administrativos em instituições de ensino, integrando funcionalidades de rastreamento de solicitações e atendimento automatizado através de inteligência artificial.

**Principais Funcionalidades:**
- Chatbot inteligente com suporte a múltiplos provedores de IA (Ollama, OpenAI, Grok)
- Dashboard administrativo com métricas em tempo real
- Sistema de autenticação seguro com criptografia bcrypt
- Interface responsiva e moderna
- Comunicação em tempo real via WebSocket
- Geração de relatórios (CSV, PDF)

**Destaques Técnicos:**
- Arquitetura cliente-servidor escalável
- Integração com múltiplos modelos de IA
- Sistema de autorização baseado em hardware
- Design responsivo e acessível
